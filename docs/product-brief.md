# FocusAnchor Product Brief

FocusAnchor helps users reduce procrastination by choosing and protecting their attention during focused work sessions.

The app is not a task manager and does not generate micro-goals.

Core loop:
1. Choose a focus intention.
2. Choose a session duration.
3. Start a focus session.
4. Capture distractions without switching tasks.
5. Review what happened.
6. Return to the next intentional focus block.

MVP features:
- Focus intention input
- Focus timer and compact floating timer
- Distraction capture during active sessions
- Session review
- Local session history
- Persistent light and dark themes
- Optional local rain sound
- Local calendars with planned focus blocks
- One optional brief daily intention per calendar
- Daily attention summary
- Optional manual Google Calendar sync

Calendar planning stays deliberately narrow:
- A plan is one focus block, not a task or a collection of steps.
- A daily goal is one brief intention for a calendar and date.
- Spontaneous focus sessions are still allowed.

Google Calendar remains optional:
- Local calendars work offline.
- One writable Google calendar can be linked to each local calendar.
- FocusAnchor plans are exported and imported as editable focus blocks.
- Other Google events are shown only as read-only context.
- Sync is manual and local changes remain pending if Google is unavailable.

Non-goals:
- No micro-goal generation
- No complex project management
- No AI coaching in the MVP
- No shame-based streaks
- No productivity scores or percentages
- No social features
